# ScriptableExchangeRateOverview
Display the history exchange rate of two currencies within a Scriptable widget using the API [https://ratesexchange.eu/](https://ratesexchange.eu/).

To use the script, download the app [Scriptable](https://scriptable.app/) on your phone and paste the script. 

> [!IMPORTANT]  
> This script is based on the exchange rate API [https://ratesexchange.eu/](https://ratesexchange.eu/). You'll need to create a free account to obtain an API key.
> This key will be stored securely using the key chain. On the first time you'll run the script, a prompt will ask for that key.

> [!CAUTION]
> This script was build heavily leveraging LLMs. No guarantees ✌️
